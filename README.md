# FastAPI CRUD Demo

This project demonstrates a simple CRUD (Create, Read, Update, Delete) API built using FastAPI and SQLAlchemy ORM.

## Features
- FastAPI framework
- SQLAlchemy ORM integration
- MySQL database
- CRUD operations (Create, Read, Update, Delete)
- Interactive API docs (Swagger UI)

## Project Structure
- main.py – API routes and application setup
- .gitignore – Ignored files configuration
